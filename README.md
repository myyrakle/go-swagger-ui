# go-swagger-ui

Embedded Swagger UI servers for Go web frameworks.

This package serves Swagger UI assets. It does not generate Swagger specs.
For generation, use tools like [swaggo](https://github.com/swaggo/swag).

## Supported Frameworks

- [echo v4](./echov4/README.md)
- [echo v5](./echov5/README.md)
- [gin v1](./ginv1/README.md)
- [fiber v2](./fiberv2/README.md)
- [fiber v3](./fiberv3/README.md)
- [net/http](./nethttp/README.md)
